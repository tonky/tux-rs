## Goal

Define the core type system that describes TUXEDO laptop hardware. Every device is a `DeviceDescriptor` with explicit capabilities and platform-specific register maps. This is the foundation all other code builds on.

## Deliverables

### Module Structure

```
tux-core/src/
├── lib.rs
├── platform.rs          # Platform enum
├── device.rs            # DeviceDescriptor + capability types
└── registers.rs         # PlatformRegisters (EC addresses, WMI GUIDs)
```

### Platform Enum

```rust
/// Hardware platform families for TUXEDO laptops.
#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash, Serialize, Deserialize)]
pub enum Platform {
    Nb05,      // EC port I/O (Pulse, InfinityFlex)
    Nb04,      // WMI AB/BS (Sirius)
    Uniwill,   // ACPI EC methods (InfinityBook)
    Clevo,     // WMI + ACPI DSM (Polaris, Stellaris)
    Tuxi,      // ACPI TFAN (Aura)
}
```

### DeviceDescriptor

```rust
pub struct DeviceDescriptor {
    pub name: &'static str,
    pub product_sku: &'static str,
    pub platform: Platform,
    pub fans: FanCapability,
    pub keyboard: KeyboardType,
    pub sensors: SensorSet,
    pub charging: ChargingCapability,
    pub tdp: Option<TdpBounds>,
    pub registers: PlatformRegisters,
}
```

### Capability Types

```rust
pub struct FanCapability {
    pub count: u8,                   // 0-3
    pub control: FanControlType,     // Direct | ProfileOnly | None
    pub pwm_scale: u8,              // max PWM value (200 for Uniwill, 255 for others)
}

pub enum FanControlType {
    Direct,      // daemon can set PWM directly
    ProfileOnly, // fans managed by firmware power profile (NB04)
    None,        // no fan control available
}

pub enum KeyboardType {
    None,
    White,                           // single-color, brightness only
    WhiteLevels(u8),                // discrete brightness levels (NB05: 3)
    Rgb1Zone,
    Rgb3Zone,
    RgbPerKey,
    IteHid(IteModel),              // USB HID via hidraw
}

pub enum IteModel {
    Ite8291,      // per-key 6x21
    Ite8291Lb,    // lightbar variants
    Ite8297,      // RGB lightbar
    Ite829x,      // per-key 6x20
}

pub struct SensorSet {
    pub cpu_temp: bool,
    pub gpu_temp: bool,
    pub fan_rpm: Vec<bool>,          // per-fan RPM availability
}

pub enum ChargingCapability {
    None,
    Flexicharger,                    // Clevo ACPI
    EcProfilePriority,              // Uniwill EC registers
}

pub struct TdpBounds {
    pub pl1_min: u32,
    pub pl1_max: u32,
    pub pl2_min: u32,
    pub pl2_max: u32,
    pub pl4_min: Option<u32>,
    pub pl4_max: Option<u32>,
}
```

pub struct ClevoRegisters {

}
