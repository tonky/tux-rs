# 2.4 — DMI Platform Detection

**Phase:** 2 — tux-core: Hardware Model System
**Estimate:** 1 day (6 hours)
**Depends on:** 2.1, 2.3
**Unlocks:** Phase 4 (daemon needs to detect hardware at startup)

## Goal

Detect the running TUXEDO laptop model at startup using DMI data and sysfs probing. Returns a resolved DeviceDescriptor from the device table, or a platform fallback if the specific SKU is unknown.

### Detection Strategy

```rust
pub struct DetectedDevice {
    pub descriptor: &'static DeviceDescriptor,
    pub dmi: DmiInfo,
    pub exact_match: bool,              // false if using platform fallback
}

pub struct DmiInfo {
    pub board_vendor: String,
    pub board_name: String,
    pub product_sku: String,
    pub sys_vendor: String,
    pub product_name: String,
    pub product_version: String,
}
```

### Detection Flow

1. Read DMI strings from `/sys/class/dmi/id/`
2. Try `lookup_by_sku(product_sku)` — exact match
3. If no match, detect platform heuristically:
   - `board_vendor == "NB05"` → Platform::Nb05
   - WMI GUID `80C9BAA6-AC48-4538-9234-9F81A55E7C85` in `/sys/bus/wmi/devices/` → Platform::Nb04
   - Sysfs `/sys/devices/platform/tuxedo-uw-fan/` exists → Platform::Uniwill
   - Sysfs `/sys/devices/platform/tuxedo-clevo/` exists → Platform::Clevo
   - Sysfs `/sys/devices/platform/tuxedo-tuxi/` exists → Platform::Tuxi
4. Use `fallback_for_platform(platform)` if platform detected but SKU unknown
5. Return error if no platform detected (not a TUXEDO laptop, or missing kernel shim)

### DmiSource Trait (testability)

```rust
pub trait DmiSource {
    fn read_dmi_field(&self, field: &str) -> io::Result<String>;
    fn wmi_guid_exists(&self, guid: &str) -> bool;
    fn sysfs_path_exists(&self, path: &str) -> bool;
}

pub struct SysFsDmiSource;    // Real implementation — reads /sys/
pub struct MockDmiSource;      // Test implementation — configurable
```

### Public API

```rust
pub fn detect_device(source: &dyn DmiSource) -> Result<DetectedDevice, DetectionError>;

pub enum DetectionError {
    NoDmiAccess(io::Error),
    UnknownPlatform { dmi: DmiInfo },
    NoKernelShim { platform: Platform },
}
```
