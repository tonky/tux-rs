# 4.1 — Fan Curve Engine

**Phase:** 4 — Fan Control + D-Bus Core
**Estimate:** 1 day (6 hours)
**Depends on:** 2.2 (FanBackend trait)
**Unlocks:** 4.2, 4.3 (backends and D-Bus wire up to engine)

## Goal

Implement the core fan curve engine: temperature polling, curve interpolation, hysteresis, minimum speed enforcement, and mode switching. This is the central control loop that keeps fans running safely.

### Fan Curve Point

```rust
// In tux-core (shared with profiles + TUI)
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct FanCurvePoint {
    pub temp: u8,     // °C
    pub speed: u8,    // 0–100%
}
```

### Interpolation

```rust
/// Linear interpolation between curve points.
/// Below first point → first speed. Above last → last speed.
/// Empty curve → 100% (safety fallback).
pub fn interpolate(curve: &[FanCurvePoint], temp: u8) -> u8;

/// Convert percentage (0–100) to PWM value (0–255).
pub fn percent_to_pwm(percent: u8) -> u8;
```

### Fan Curve Engine

```rust
pub struct FanCurveEngine {
    backend: Arc<dyn FanBackend>,
    config_rx: watch::Receiver<FanConfig>,
}

pub struct FanConfig {
    pub mode: FanMode,
    pub min_speed_percent: u8,          // default: 25
    pub curve: Vec<FanCurvePoint>,
    pub active_poll_ms: u64,            // default: 2000
    pub idle_poll_ms: u64,              // default: 10000
    pub hysteresis_degrees: u8,         // default: 3
}

pub enum FanMode {
    Auto,         // Restore hardware auto control
    Manual,       // No daemon intervention (static user-set PWM)
    CustomCurve,  // Poll temp → interpolate → write PWM
}
```

### Engine Run Loop

```rust
impl FanCurveEngine {
    pub async fn run(&mut self, mut shutdown: broadcast::Receiver<()>);
}
```

**Loop logic (CustomCurve mode):**
1. Read temperature via `backend.read_temp()`
2. Check hysteresis: if `|temp - last_temp| < hysteresis_degrees`, skip update
3. Interpolate target speed from curve
4. Apply min speed floor: `speed = speed.max(min_speed_percent)`
5. Convert to PWM: `pwm = percent_to_pwm(speed)`
6. Write to all fans: `backend.write_pwm(i, pwm)` for i in 0..num_fans
7. Sleep for poll interval
8. On config change (watch channel): reload curve/mode immediately

**Auto mode:** Call `backend.set_auto(i)` for all fans, then stop polling.
**Manual mode:** No-op loop (just watch for config changes).
