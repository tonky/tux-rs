# 4.3 — D-Bus Server + systemd

**Phase:** 4 — Fan Control + D-Bus Core
**Estimate:** 1 day (6 hours)
**Depends on:** 4.1, 4.2
**Unlocks:** 4.4, 5.3, Phase 6 (TUI connects to this D-Bus API)

## Goal

Stand up the D-Bus server on `com.tuxedocomputers.tccd` with fan control and device info interfaces. Integrate with systemd (Type=dbus, watchdog, sd_notify). Load config from TOML.

### D-Bus Interfaces

**Bus name:** `com.tuxedocomputers.tccd`
**Object path:** `/com/tuxedocomputers/tccd`

#### Fan Interface (`com.tuxedocomputers.tccd.Fan`)

| Method | Signature | Description |
|--------|-----------|-------------|
| `SetFanSpeed` | `(fan_index: u32, pwm: u8)` | Write PWM directly |
| `SetAutoMode` | `(fan_index: u32)` | Restore hardware auto |
| `GetFanSpeed` | `(fan_index: u32) → u32` | Read RPM |
| `GetTemperature` | `(sensor_index: u32) → i32` | Read temp (millidegrees) |
| `GetFanInfo` | `() → (u32, u32, bool, u8)` | (max_rpm, min_rpm, multi_fan, num_fans) |
| `SetFanMode` | `(mode: &str)` | "auto" / "manual" / "custom" |
| `SetFanCurve` | `(toml: &str)` | TOML-encoded fan curve |
| `GetActiveFanCurve` | `() → String` | Current curve as TOML |

**Properties:**
- `FanCount: u32`

**Signals:**
- `FanSpeedChanged(fan_index: u32, rpm: u32)`
- `TemperatureChanged(sensor_index: u32, millidegrees: i32)`

#### Device Interface (`com.tuxedocomputers.tccd.Device`)

**Properties:**
- `DeviceName: String`
- `Platform: String`
- `DaemonVersion: String`

### systemd Service

```ini
[Unit]
Description=TUXEDO Hardware Daemon
After=dbus.service
Requires=dbus.service

[Service]
Type=dbus
BusName=com.tuxedocomputers.tccd
ExecStart=/usr/bin/tux-daemon
Restart=on-failure
RestartSec=2s
WatchdogSec=30s
NotifyAccess=main

[Install]
WantedBy=multi-user.target
```

### D-Bus Policy File

```xml
<!DOCTYPE busconfig PUBLIC "-//freedesktop//DTD D-BUS Bus Configuration 1.0//EN"
 "http://www.freedesktop.org/standards/dbus/1.0/busconfig.dtd">
<busconfig>
  <policy user="root">
    <allow own="com.tuxedocomputers.tccd"/>
  </policy>
  <policy context="default">
    <allow send_destination="com.tuxedocomputers.tccd"/>
    <allow receive_sender="com.tuxedocomputers.tccd"/>
  </policy>
</busconfig>
```

### Startup Sequence

1. Initialize tracing (log level from config or env)
2. Load config from `/etc/tux-daemon/config.toml` (fall back to defaults)
3. Detect device via `detect_device()`
4. Initialize fan backend via `init_fan_backend()`
5. Create `watch::channel` for fan config
6. Spawn `FanCurveEngine::run()` as background task
7. Register D-Bus service on system bus
8. `sd_notify::notify(Ready)`
9. Spawn watchdog ping task (every `WatchdogSec/2`)
10. Wait for shutdown signal
