# 4.4 — Safety + Graceful Shutdown

**Phase:** 4 — Fan Control + D-Bus Core
**Estimate:** 1 day (6 hours)
**Depends on:** 4.1, 4.2, 4.3
**Unlocks:** Phase 5, Phase 6

## Goal

Ensure fans are always safe: auto-restore on shutdown/crash, active/idle polling modes, and integration testing of the full daemon lifecycle.

## Deliverables

### Graceful Shutdown

```rust
// Signal handler in main.rs
async fn shutdown_handler(
    fan_engine: JoinHandle<()>,
    fan_backend: Arc<dyn FanBackend>,
) {
    // Wait for SIGTERM or SIGINT
    tokio::select! {
        _ = signal::ctrl_c() => {},
        _ = sigterm_future() => {},
    }

    // 1. Notify systemd
    sd_notify::notify(false, &[NotifyState::Stopping]);

    // 2. Abort fan curve engine
    fan_engine.abort();

    // 3. Restore all fans to auto mode
    for i in 0..fan_backend.num_fans() {
        let _ = fan_backend.set_auto(i);
    }

    // 4. D-Bus connection drops automatically
}
```

### Auto-Restore on Crash

- systemd `Restart=on-failure` + `RestartSec=2s` ensures daemon restarts
- On startup, `init_fan_backend()` calls `set_auto()` for all fans before any custom curve is applied
- This means even after a crash, fans return to hardware auto within 2 seconds

### Active/Idle Polling Modes

```rust
pub struct ClientTracker {
    active_clients: AtomicUsize,
}

impl ClientTracker {
    // Track via D-Bus NameOwnerChanged signal
    pub fn on_client_connected(&self);
    pub fn on_client_disconnected(&self);
    pub fn has_active_clients(&self) -> bool;
    pub fn current_poll_interval(&self, config: &FanConfig) -> Duration;
}
```

| State | Poll Interval | D-Bus Signals | Fan Control |
|-------|--------------|---------------|-------------|
| Active (≥1 client) | `active_poll_interval_ms` (2s) | Emit temp + RPM signals | Full curve engine |
| Idle (0 clients) | `idle_poll_interval_ms` (10s) | No signals emitted | Hold last curve speed |

**Transition logic:**
- Client connects → switch to active immediately
- Client disconnects → start `idle_timeout_s` (30s) grace period → switch to idle
- Grace period prevents oscillation from TUI open/close cycles

### Watchdog Integration

```rust
async fn watchdog_task() {
    let interval = sd_notify::watchdog_interval()
        .unwrap_or(Duration::from_secs(30)) / 2;

    loop {
        sd_notify::notify(false, &[NotifyState::Watchdog]);
        tokio::time::sleep(interval).await;
    }
}
```
