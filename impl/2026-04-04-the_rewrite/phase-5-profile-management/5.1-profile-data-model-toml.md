## Goal

Define the profile data model, implement TOML serialization, and build the ProfileStore for CRUD operations. Profiles persist as individual TOML files in `/etc/tux-daemon/profiles/`.

## Deliverables

```
└── ...

tux-daemon/src/

└── ...
```

### TuxProfile

```rust
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct TuxProfile {
    pub id: String,                      // unique identifier
    pub name: String,                    // display name
    pub description: String,
    pub is_default: bool,                // read-only built-in profiles
    pub fan: FanProfileSettings,
    pub cpu: CpuSettings,
    pub keyboard: KeyboardSettings,
    pub display: DisplaySettings,
    pub charging: ChargingSettings,
    pub odm_profile: Option<String>,     // "powersave" / "balanced" / "performance"
    pub tdp: Option<TdpSettings>,
}
```

### Sub-structs

```rust
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct FanProfileSettings {
    pub enabled: bool,
    pub mode: FanMode,
    pub min_speed_percent: u8,
    pub max_speed_percent: u8,
    pub curve: Vec<FanCurvePoint>,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct CpuSettings {
    pub governor: String,                 // "powersave" / "performance" / "schedutil"
    pub energy_performance_preference: Option<String>,  // "power" / "balance_power" / "balance_performance" / "performance"
    pub no_turbo: bool,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct KeyboardSettings {
    pub brightness: u8,                   // 0–100
    pub color: String,                    // hex "#rrggbb"
    pub mode: String,                     // "static" / "breathe" / "rainbow"
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct DisplaySettings {
    pub brightness: Option<u8>,           // 0–100
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct ChargingSettings {
    pub profile: Option<String>,          // "high_capacity" / "balanced" / "stationary"
    pub priority: Option<String>,
    pub start_threshold: Option<u8>,      // 0–100%
    pub end_threshold: Option<u8>,        // 0–100%
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct TdpSettings {
    pub pl1: Option<u32>,
    pub pl2: Option<u32>,
    pub pl4: Option<u32>,
}
```

### ProfileStore

```rust
pub struct ProfileStore {
    dir: PathBuf,                         // /etc/tux-daemon/profiles/
    profiles: HashMap<String, TuxProfile>,
}

impl ProfileStore {
    pub fn load(dir: impl Into<PathBuf>) -> Result<Self>;
    pub fn list(&self) -> Vec<&TuxProfile>;
    pub fn get(&self, id: &str) -> Option<&TuxProfile>;
    pub fn create(&mut self, profile: TuxProfile) -> Result<String>;
    pub fn update(&mut self, id: &str, profile: TuxProfile) -> Result<()>;
    pub fn delete(&mut self, id: &str) -> Result<()>;
    pub fn copy(&mut self, id: &str) -> Result<String>;
    fn save_to_disk(&self, profile: &TuxProfile) -> Result<()>;
    fn delete_from_disk(&self, id: &str) -> Result<()>;
}
```

**TOML file format** (`/etc/tux-daemon/profiles/my-profile.toml`):
```toml
id = "my-custom-profile"
name = "My Custom Profile"
description = "Balanced for daily use"
is_default = false

[fan]
enabled = true
mode = "custom-curve"
min_speed_percent = 20
max_speed_percent = 100

[[fan.curve]]
temp = 40
speed = 0

[[fan.curve]]
temp = 70
speed = 50

[[fan.curve]]
temp = 90
speed = 100

[cpu]
governor = "schedutil"
no_turbo = false

[keyboard]
brightness = 50
color = "#ffffff"
mode = "static"
```
