# 6.1 — TEA Architecture + Shell

**Phase:** 6 — Terminal Interface
**Estimate:** 1 day (6 hours)
**Depends on:** 4.3 (D-Bus API available)
**Unlocks:** 6.2, 6.3, 6.4, 6.5

## Goal

Set up the TUI application skeleton using the TEA (The Elm Architecture) pattern. Tab navigation, D-Bus connection, event loop, and basic rendering. After this sub-phase, the TUI compiles, renders a tab bar, and connects to the daemon.

## Deliverables

### Module Structure

```
tux-tui/src/
├── main.rs              # Entry point, terminal setup, event loop
```

### TEA Pattern

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│   Events    │ ──→ │    Update    │ ──→ │    View     │
│ (keys,data) │     │ (pure state) │     │  (render)   │
└─────────────┘     └──────────────┘     └─────────────┘
       ↑                    │                     │
       │                    ↓                     ↓
       │              ┌──────────┐          ┌──────────┐
       └───────────── │ Commands │          │ Terminal  │
                      │ (D-Bus)  │          │ (stdout)  │
                      └──────────┘          └──────────┘
```

### Event Loop

```rust
pub enum AppEvent {
    Key(KeyEvent),
    Resize(u16, u16),
    DbusData(DbusUpdate),
    Tick,
}

// Main loop
loop {
    // Render
    terminal.draw(|f| view::render(f, &model))?;

    // Wait for event
    match event_rx.recv().await? {
        AppEvent::Key(key) => {
            let commands = update::handle_key(&mut model, key);
            for cmd in commands { execute_command(cmd, &dbus).await; }
        }
        AppEvent::DbusData(data) => update::handle_data(&mut model, data),
        AppEvent::Tick => { /* periodic refresh */ }
        AppEvent::Resize(w, h) => model.terminal_size = (w, h),
    }

    if model.should_quit { break; }
}
```

### View Rendering

```rust
pub fn render(frame: &mut Frame, model: &Model) {
    let chunks = Layout::default()
        .direction(Direction::Vertical)
        .constraints([
            Constraint::Length(1),    // Tab bar
            Constraint::Min(0),      // Tab content
            Constraint::Length(1),    // Status bar
        ])
        .split(frame.area());

    render_tab_bar(frame, chunks[0], model);
    render_tab_content(frame, chunks[1], model);
    render_status_bar(frame, chunks[2], model);

    if model.show_help {
        render_help_overlay(frame, model);
    }
}
```
