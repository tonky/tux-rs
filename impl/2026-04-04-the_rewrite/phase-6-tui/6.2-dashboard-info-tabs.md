**Components:**
- Fan gauges: horizontal bar with color zones (green вүӨ40%, yellow вүӨ70%, red >70%)
- Sparkline history: 60-point rolling buffer for temp and fan speed
- CPU info line: temperature, average frequency, core count
- Power state: AC (вҡЎ) or Battery (рҹ”Ӣ)
- Active profile name

**Data Model:**
```rust
pub struct DashboardState {
    pub fan_speeds: Vec<FanData>,       // per fan: rpm, pwm percent
    pub cpu_temp: f32,                  // В°C
    pub cpu_freq_mhz: u32,
    pub core_count: u32,
    pub power_state: PowerState,
    pub active_profile: String,
    pub temp_history: VecDeque<f32>,    // 60 points
    pub speed_history: VecDeque<f32>,   // 60 points
}

pub struct FanData {
    pub rpm: u32,
    pub speed_percent: u8,
}
```

**Polling:**
- Fast interval: 1 second for dashboard telemetry
- D-Bus calls: `GetTemperature`, `GetFanSpeed`, `GetFanInfo`, `GetPowerState`
