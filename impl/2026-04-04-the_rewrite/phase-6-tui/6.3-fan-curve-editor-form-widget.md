**Datasets (4):**
1. Curve line — connected path through all points
2. All points — dots at each FanCurvePoint
3. Current operating point — yellow dot showing live temp/speed
4. Selected point — magenta highlight for editing

**Editing keys:**
| Key | Action |
|-----|--------|
| `←` / `→` | Select previous/next point |
| `↑` / `↓` | Adjust selected point speed ±5% |
| `i` | Insert new point between selected and next |
| `x` | Delete selected point (min 2 points) |
| `r` | Reset to default 4-point curve |
| `s` | Save curve to daemon via D-Bus |
| `Esc` | Discard changes |

**State:**
```rust
pub struct FanCurveState {
    pub points: Vec<FanCurvePoint>,       // editable copy
    pub selected_index: usize,
    pub current_temp: u8,                 // live from daemon
    pub current_speed: u8,                // live from daemon
    pub dirty: bool,                      // unsaved changes
    pub original_points: Vec<FanCurvePoint>, // for reset
}
```

### Generic Form Widget

A reusable form widget for settings tabs. Renders a list of labeled fields with type-specific editing.

```rust
pub struct Form {
    pub fields: Vec<FormField>,
    pub selected_index: usize,
    pub dirty: bool,
}

pub struct FormField {
    pub label: String,
    pub field_type: FieldType,
    pub enabled: bool,
}

pub enum FieldType {
    Text(String),                          // Free text input
    Number { value: i64, min: i64, max: i64, step: i64 },
    Bool(bool),                            // Toggle
    Select { options: Vec<String>, selected: usize },
}
```

**Rendering:**
```
┌─ Settings ────────────────────────────────┐
│                                           │
│  Temperature Unit    [▸ Celsius  ]        │  ← Select
│  Fan Control         [✓]                  │  ← Bool
│  CPU Settings        [✓]                  │  ← Bool
│  Min Fan Speed       [  25 ] %            │  ← Number
│                                           │
│  [s] Save   [Esc] Discard                 │
└───────────────────────────────────────────┘
```

**Form keys:**
| Key | Action |
|-----|--------|
| `↑` / `↓` | Navigate fields |
| `←` / `→` | Adjust value (Number: ±step, Select: prev/next, Bool: toggle) |
| `Space` | Toggle bool, cycle select |
| `Enter` | Edit text field (inline) |
| `s` | Save (emit Command) |
| `Esc` | Discard changes |
