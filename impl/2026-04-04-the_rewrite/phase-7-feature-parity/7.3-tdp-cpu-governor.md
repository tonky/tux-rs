# 7.3 — TDP + CPU Governor Control

**Phase:** 7 — Feature Parity
**Estimate:** 1 day (6 hours)
**Depends on:** 5.1 (profile model includes CPU/TDP settings)
**Unlocks:** Independent

## Goal

Control CPU power limits (TDP: PL1/PL2/PL4 on Uniwill) and CPU governor/EPP settings via sysfs. These settings are applied as part of profile switching.

### TDP Control (Uniwill)

```rust
pub trait TdpBackend: Send + Sync {
    fn get_pl1(&self) -> io::Result<u32>;     // watts
    fn set_pl1(&self, watts: u32) -> io::Result<()>;
    fn get_pl2(&self) -> io::Result<u32>;
    fn set_pl2(&self, watts: u32) -> io::Result<()>;
    fn bounds(&self) -> &TdpBounds;           // from DeviceDescriptor
}

pub struct UniwillTdp {
    ec: EcRam,    // pread/pwrite to ec_ram binary attribute
    bounds: TdpBounds,
}
```

Uniwill EC registers for TDP (from tuxedo-drivers-rs):
- PL1, PL2 at known EC offsets
- Values clamped to `TdpBounds` from DeviceDescriptor

### Profile Application Integration

In `ProfileApplier`:
```rust
pub async fn apply(&self, profile: &TuxProfile, device: &DetectedDevice) -> Result<()> {
    // Fan curve (already implemented in 5.2)
    self.fan_config_tx.send(FanConfig::from(&profile.fan))?;

    // CPU governor
    if let Some(ref cpu) = self.cpu_governor {
        cpu.set_governor(&profile.cpu.governor)?;
        if let Some(ref epp) = profile.cpu.energy_performance_preference {
            cpu.set_epp(epp)?;
        }
        cpu.set_no_turbo(profile.cpu.no_turbo)?;
    }

    // TDP
    if let (Some(ref tdp_backend), Some(ref tdp_settings)) = (&self.tdp_backend, &profile.tdp) {
        if let Some(pl1) = tdp_settings.pl1 {
            tdp_backend.set_pl1(pl1)?;
        }
        if let Some(pl2) = tdp_settings.pl2 {
            tdp_backend.set_pl2(pl2)?;
        }
    }

    Ok(())
}
```
