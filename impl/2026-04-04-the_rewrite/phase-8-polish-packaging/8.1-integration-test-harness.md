# 8.1 — Integration Test Harness

**Phase:** 8 — Polish & Packaging
**Estimate:** 1 day (6 hours)
**Depends on:** Phase 5 (profiles), Phase 6 (TUI)
**Unlocks:** 8.2 (packaging), 8.3 (documentation)

## Goal

Build an integration test harness that exercises the full daemon+TUI stack against a mock sysfs tree. Tests must run without root, without hardware, and in CI (headless).

## Deliverables

### Mock Sysfs Tree

```rust
pub struct MockSysfs {
    root: TempDir,
}

impl MockSysfs {
    pub fn new() -> Self;

    /// Create mock fan hwmon device
    pub fn add_fan(&self, name: &str, num_fans: u8) -> PathBuf;

    /// Create mock CPU governor sysfs
    pub fn add_cpu(&self, cores: u8) -> PathBuf;

    /// Create mock battery/charging sysfs
    pub fn add_battery(&self, name: &str) -> PathBuf;

    /// Create mock GPU hwmon
    pub fn add_gpu(&self, name: &str) -> PathBuf;

    /// Create mock EC binary attribute
    pub fn add_ec_ram(&self) -> PathBuf;

    /// Create mock hidraw device
    pub fn add_hidraw(&self, vid: u16, pid: u16) -> PathBuf;

    /// Root path — passed to daemon as sysfs_root override
    pub fn root(&self) -> &Path;
}
```

### Full Stack Test

```rust
#[tokio::test]
async fn test_daemon_tui_roundtrip() {
    let sysfs = MockSysfs::new();
    sysfs.add_fan("test-fan", 2);
    sysfs.add_cpu(4);

    // Start daemon with mock sysfs root and session bus
    let daemon = TestDaemon::start(&sysfs).await;

    // D-Bus call: create profile
    daemon.dbus_call("SetProfile", test_profile_toml()).await;

    // Verify: fan PWM files written
    assert_eq!(sysfs.read_fan_pwm("test-fan", 0), 128);

    // D-Bus call: get profile
    let result = daemon.dbus_call("GetProfile", "test").await;
    assert!(result.contains("test"));

    daemon.stop().await;
}
```

### TestDaemon Helper

```rust
pub struct TestDaemon {
    child: JoinHandle<()>,
    bus_name: String,
    connection: zbus::Connection,
}

impl TestDaemon {
    /// Start daemon on private session bus
    pub async fn start(sysfs: &MockSysfs) -> Self;

    /// Make a D-Bus method call
    pub async fn dbus_call(&self, method: &str, args: &str) -> String;

    /// Graceful shutdown
    pub async fn stop(self);
}
```

### Headless TUI Tests

```rust
#[test]
fn test_tui_renders_dashboard() {
    let backend = TestBackend::new(120, 40);
    let mut terminal = Terminal::new(backend).unwrap();
    let mut app = App::new_test(mock_dbus_state());

    terminal.draw(|f| app.render(f)).unwrap();

    let buffer = terminal.backend().buffer();
    // Verify expected text appears
    assert!(buffer_contains_text(buffer, "Dashboard"));
    assert!(buffer_contains_text(buffer, "CPU"));
}
```

### CI Configuration

GitHub Actions workflow that:
1. `cargo test --workspace` — all unit tests
2. `cargo test --workspace -- --ignored` — integration tests
3. `cargo clippy --workspace -- -D warnings`
4. `cargo fmt --check`
