### Migration Guide (`MIGRATING.md`)

For users switching from tuxedo-drivers + tuxedo-control-center:

1. **Uninstall old packages:**
   ```bash
   sudo apt remove tuxedo-drivers tuxedo-control-center
   ```

2. **Install tux-rs:**
   ```bash
   sudo dpkg -i tux-rs_0.1.0_amd64.deb
   ```

3. **Config migration:**
   - Old: `/etc/tcc/profiles/*.json` → New: `/etc/tux-daemon/profiles/*.toml`
   - Provide conversion script: `tux-migrate-config`
   - Maps JSON profile fields to TOML equivalents

4. **Behavioral differences:**
   - Single binary daemon (no separate keyboard/fan services)
   - TOML config (not JSON)
   - Profiles in `/etc/tux-daemon/profiles/` (not per-user)
   - TUI replaces Qt GUI

5. **Troubleshooting:**
   - Check `journalctl -u tux-daemon`
   - Verify kernel shims loaded: `lsmod | grep tuxedo`
   - D-Bus introspection: `busctl introspect com.tuxedocomputers.tccd`
