# 8.2 — Packaging

**Phase:** 8 — Polish & Packaging
**Estimate:** 1 day (6 hours)
**Depends on:** 8.1 (integration tests pass)
**Unlocks:** 8.3 (documentation)

## Goal

Create distribution packages (deb, rpm, PKGBUILD) and a Justfile with install/uninstall targets. Users can install tux-rs from packages or `just install`.

## Deliverables

### Justfile

```just
# Build
build:
    cargo build --release

# Install everything: daemon, TUI, kernel shims, systemd, D-Bus, config
install: build
    install -Dm755 target/release/tux-daemon  /usr/bin/tux-daemon
    install -Dm755 target/release/tux-tui     /usr/bin/tux-tui
    install -Dm644 dist/tux-daemon.service    /etc/systemd/system/tux-daemon.service
    install -Dm644 dist/com.tuxedocomputers.tccd.conf /usr/share/dbus-1/system.d/com.tuxedocomputers.tccd.conf
    install -Dm644 dist/tux-daemon.toml       /etc/tux-daemon/config.toml
    # Kernel shims
    @for mod in tux-kmod/shims/*/; do \
        make -C $$mod install; \
    done
    systemctl daemon-reload
    systemctl enable --now tux-daemon

uninstall:
    systemctl disable --now tux-daemon || true
    rm -f /usr/bin/tux-daemon /usr/bin/tux-tui
    rm -f /etc/systemd/system/tux-daemon.service
    rm -f /usr/share/dbus-1/system.d/com.tuxedocomputers.tccd.conf
    # Kernel shims
    @for mod in tux-kmod/shims/*/; do \
        make -C $$mod uninstall; \
    done
    systemctl daemon-reload

# Run checks
check:
    cargo test --workspace
    cargo clippy --workspace -- -D warnings
    cargo fmt --check
```

### Distribution Files (`dist/`)

```
dist/
├── tux-daemon.service              # systemd unit
├── com.tuxedocomputers.tccd.conf   # D-Bus policy
├── tux-daemon.toml                 # Default config
├── postinst.sh                     # Post-install: systemctl enable
├── prerm.sh                        # Pre-remove: systemctl stop
└── completions/
    ├── tux-tui.bash
    ├── tux-tui.zsh
    └── tux-tui.fish
```

### Debian Package

- Use `cargo-deb` for generating `.deb`
- `Cargo.toml` metadata:
  ```toml
  [package.metadata.deb]
  maintainer = "TUXEDO Computers"
  section = "utils"
  depends = "$auto"
  assets = [...]
  systemd-units = { unit-name = "tux-daemon" }
  ```

### RPM Package

- Use `cargo-generate-rpm`
- RPM spec metadata in `Cargo.toml`:
  ```toml
  [package.metadata.generate-rpm]
  assets = [...]
  ```

### Arch PKGBUILD

```bash
pkgname=tux-rs
pkgver=0.1.0
pkgrel=1
arch=('x86_64')
depends=('dkms')
makedepends=('rust' 'cargo' 'linux-headers')

build() {
    cargo build --release
    # Build kernel shims
}

package() {
    install -Dm755 target/release/tux-daemon "$pkgdir/usr/bin/tux-daemon"
    install -Dm755 target/release/tux-tui "$pkgdir/usr/bin/tux-tui"
    install -Dm644 dist/tux-daemon.service "$pkgdir/usr/lib/systemd/system/tux-daemon.service"
    # ...
}
```

### DKMS for Kernel Shims

```
dkms.conf:
PACKAGE_NAME="tux-kmod"
PACKAGE_VERSION="0.1.0"
BUILT_MODULE_NAME[0]="tuxedo_ec"
BUILT_MODULE_NAME[1]="tuxedo_uw_fan"
...
DEST_MODULE_LOCATION[0]="/updates"
AUTOINSTALL="yes"
```
